1. Дан код: var a = 1, b = 1, c, d;
c = ++a; alert(c); // 2 - префиксный оператор ++ :сначала к a прибавляется единица, затем вычисляется значение c
d = b++; alert(d); // 1 - постфиксный оператор ++ :сначала вычисляется значение d, затем к b прибавляется единица и записывается (b=2)
c = (2+ ++a); alert(c); // 5 - префиксный оператор ++ : к 2 прибавляем а, увеличенную на единицу, а = 2 с прошлого цикла, итого а = 3
d = (2+ b++); alert(d); // 4 - постфиксный оператор ++: сначала к 2 прибавляется b (с прошлого цикла b=2), затем b увеличивается на единицу b и записывается (b=3)
alert(a); // 3 - a с предыдущего цикла равно 3
alert(b); // 3 - b с предыдущего цикла равно 3
Почему код даёт именно такие результаты?
